What It Is

Large open-source tactical strategy game cached from Debian packages with no-internet launch smoke.

Game/project page: https://packages.debian.org/bookworm/ufoai

Quick Manual

  1. Start a campaign or skirmish from the local client.
  2. Use geoscape/base screens to prepare, then tactical missions for combat.
  3. Large install; keep it in the full GannanNet profile rather than Raspberry Pi profile.
