Unciv LAN Server

A Civilization-style progression candidate: each device runs Unciv, while the VM or Pi stores multiplayer turn files over HTTP.

Server URL

Use this multiplayer server URL in the Unciv client:

http://192.168.1.106:8090

Health check endpoint: /isalive.

Activation

The Docker image is cached. Start conservatively for Pi-style testing:

UNCIV_JAVA_OPTS="-Xms64m -Xmx256m" docker compose -f deploy/unciv.compose.yml up -d

Offline Checklist

  • Cache Unciv APK/desktop release before leaving internet.
  • Run one full create-game, upload-turn, download-turn loop.
  • Keep server data under /var/lib/lan-arcade/unciv.
