What It Is

FreeRCT aims to provide a free and open source theme-park management game in the spirit of the classic RollerCoaster Tycoon games. This release is early, so treat it as an experimental tycoon option rather than a finished replacement.

Source: https://freerct.net/

Quick Manual

  1. Install the local Debian package from the shelf on a Linux client or run the VM-installed client for testing.
  2. Start a park, inspect build menus, and try paths, rides, shops, and entrances.
  3. Ready status needs a placed path or ride; current check is launch-only partial.
