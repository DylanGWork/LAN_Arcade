How To Play

  • Open the DOS shelf to review status, screenshots, and source notes. Browser Play buttons return after the private package source cache is restored.
  • Click inside the emulator before using the keyboard or mouse.
  • After packages are restored, if the browser blocks the threaded DOSBox core, use the package download on the same page with a local DOSBox install.
  • Native EXE downloads are not the first path for these DOS games; the intended path is a VM-generated local browser package.
