# Gene Garden Source

Gene Garden is an original LAN Arcade artificial-life game.

It was built as a clean-room, first-party game inspired by the broader genre of
evolving critter and genetics sandboxes. It does not copy Some-Life code,
assets, layout, or text.

Core ideas in this copy:

- Grid-based grass growth and seasonal pressure.
- Critters with inherited numeric genes.
- Mutation, reproduction, family lines, and generation tracking.
- Player tools for grass, walls, erasing, adding critters, storms, and layouts.
- Offline-only HTML, CSS, and JavaScript with no external runtime dependencies.
