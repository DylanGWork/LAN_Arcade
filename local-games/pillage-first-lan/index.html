Should You Play?

Pick this when you want the slow strategy loop of villages, resources, upgrades, timing, and raids without an always-online account or a competitive server schedule.

What it is

A browser strategy game where each world is stored on your device. You can create multiple worlds and return later as queues, troops, and events continue. The live game uses HTTPS because browser OPFS storage is blocked on plain HTTP LAN IPs.

Why it fits LAN Arcade

It is lightweight, web-based, open source, and designed to be offline-first. This is the only live game from the Travian-like batch.

What changed for LAN

The mirror is patched for /mirrors/pillage-first/ hosting and does not start automatic PeerJS discovery on page load.

Quick Manual

The game has its own in-app flow, but these are the first-session decisions that matter.

Create a world

Use the launch button, choose Create new world, then pick the basic settings. Start conservative while you learn the queues.

Stabilize resources

Upgrade resource fields and avoid spending everything on one path. Growth is easier when production stays balanced.

Build the village center

Add buildings that unlock training, storage, research, and troop movement. Queues continue after you leave the page.

Train and raid carefully

Troops are useful, but bad timing can stall your economy. Treat early raids as tests, not the whole game.
